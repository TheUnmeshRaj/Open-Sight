# Spatio-Temporal Crime Hotspot Prediction

This project predicts crime hotspots using spatio-temporal deep learning models.
It leverages LSTM-based architectures to capture temporal crime patterns and
generate predictive heatmaps for decision support.

## Features
- Spatio-temporal crime data preprocessing
- LSTM-based crime forecasting
- Hotspot visualization
- Modular and extensible architecture

## Tech Stack
- Python
- PyTorch
- NumPy, Pandas
- Jupyter Notebook

## Usage
1. Preprocess data
2. Train model
3. Generate predictions
4. Visualize hotspots
